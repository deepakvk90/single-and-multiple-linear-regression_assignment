# single-and-multiple-linear-regression_assignment
Machine Learning project demonstrating Single Variable and Multiple Linear Regression using Python and Scikit-Learn to predict daily temperature based on weather conditions.
Project Overview

This project demonstrates the implementation of Linear Regression using Python and Scikit-Learn. The objective is to predict daily temperature based on weather-related features.

The project covers:

Single Variable Linear Regression
Multiple Linear Regression
Model Training
Prediction using trained models
Understanding coefficients and intercepts
Dataset

The dataset contains weather-related information:

Feature	Description
hours_sunlight	Total hours of sunlight received in a day
humidity_level	Humidity percentage
daily_temperature	Daily temperature in °C
